## 🚀 Quick Start

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the System**:
   ```bash
   python main.py
   ```

3. **Web Interface**:
   ```bash
   python main.py --web
   ```
